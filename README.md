# Sabor da Terra - Armazém

Este é o diretório de desenvolvimento do site **Sabor da Terra - Armazém**, uma loja de produtos naturais de Mangaratiba/RJ focada em saúde integrativa, temperos finos, grãos, sementes, ervas e chás selecionados, integrada a uma enciclopédia botânica interativa de alta fidelidade (Herbário).

## Estrutura de Arquivos

- `index.html`: Estrutura semântica e SEO do site (inclui dados reais de contato e localização).
- `index.css`: Sistema de design premium (paleta inspirada no cartão de visita: Verde Floresta Profundo e Lime/Vibrant Green, suporte a Dark Mode e transições fluidas).
- `app.js`: Lógica interativa (busca, filtros, carrinho de compras, quiz e cronômetro de infusão).
- `assets/`: Imagens conceituais geradas sob demanada.

*Recomendamos definir esta pasta como seu workspace ativo no editor para facilitar o desenvolvimento!*
